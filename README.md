# FANSOFT TAKEAWAY ASSIGNMENT

Requested to create a program to add new customers.

## Setup Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd fansoft
```

## Backend

1. `cd backend`
2. `yarn install`
3. `yarn start`

## Frontend

1. `cd frontend`
2. `yarn install`
3. `yarn dev`

## Tech Stack

**Client:** React, Redux, Bootstrap 5

**Server:** Node, Express, MongoDB
